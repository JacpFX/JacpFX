What it is...

JacpFX is a Rich Client framework on top of JavaFX8.

Loosely coupled...

JacpFX components are referenced by id.

Structuring and composition...

JacpFX components are simple controller classes. Compose your applications with FXML and plain JavaFX.

Messaging...

All components connected through a message bus. Change components state through messages.

Non-Blocking UI...

JacpFX components have a defined LifeCycle to separate the task execution and UI changes.
